Website bisnis cookies untuk presentasi BizClash.

## Isi File

- `index.html`  
  Website utama KUMA dalam satu file, siap dibuka langsung di browser.

- `README.md`  
  Petunjuk penggunaan, struktur konten, dan catatan aset gambar.

## Cara Pakai

1. Simpan file `index.html` dan `README.md` dalam satu folder.
2. Buat folder gambar:
   - `public/images/products`
3. Masukkan foto produk asli Anda ke folder tersebut.
4. Ganti placeholder gambar di bagian hero dan produk jika perlu.
5. Buka `index.html` langsung di browser.

## Catatan Gambar

Website ini memakai placeholder teks:
- `Replace this image with actual product photo I send`

Silakan ganti dengan foto produk asli yang sudah dibersihkan background-nya menjadi transparan atau sesuai kebutuhan presentasi.
